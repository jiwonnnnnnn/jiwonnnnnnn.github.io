소개딥페이크 영상 판별이라는 주제로 진행된 인공지능 경진대회입니다.

대회 소개

  • 국방 분야에서 활용성이 높은 문제를 출제하고, AI 기술을 활용해 문제를 해결하도록 하는 대회
  • 부문별 출제된 과제와 제공 데이터를 활용하여 문제를 해결하도록 하고, 실시간으로 리더보드에 순위 반영 및 현장 공개

과제 및 데이터 소개

  • 지능형 사이버 위협 대응을 위한 딥페이크 영상 판별
  • Train Data
    • 약 7000개의 영상 데이터(딥페이크 영상 3500개, 실제 영상 3500개)
  • Test Data
    • 딥페이크 영상과 실제 영상이 섞인 데이터(공개x)

과제 방향성

  1. Classification
    • 데이터를 input으로 받으면 딥페이크인지 아닌지 구별해야 하므로 classification 모델을 사용해야 한다고 판단
  1. 데이터의 특성 및 처리
    • 7000개의 train 영상 데이터의 길이가 각기 다르므로 알맞은 처리가 필요할 것이라고 판단
  1. 얼굴 탐지
    • 딥페이크는 AI를 기반으로 하여 이미지에서 사람의 얼굴 부위를 합성하는 기술이므로, 중요 feature가 전체 이미지 중 일부분(얼굴이 있는 부분)에 집중되어 있음
    • 제공된 영상이 모두 사람이 중앙에서 정면을 바라보고 있었기 때문에 얼굴이 있는 부분을 정확하게 탐지해서 추출하는 작업이 필요할 것이라고 판단

모델 구성

  1. Facial Detection
  • 입력 데이터에서 얼굴 부분을 탐지하여 특징을 추출할 수 있도록 Facial Detection 모델인 MTCNN을 활용
    • MTCNN 모델은 joint learning 방법을 사용하여 다른 모델에 비해 정확도가 높고 속도가 빠르다고 알려져 있음
    • 비디오 데이터에서 3프레임 간격으로 20장의 이미지를 추출
    • MTCNN을 이용해 얼굴 부분을 crop하고 모델의 input으로 사용
    • 이때, 안면 비율을 유지하도록 하면서 margin을 20% 포함하여 resize 해줌으로써 robust하게 학습되도록 함

  1. Classification
  • EfficientNet과 Vision Transformer의 구조를 결합한 모델 사용
  • CNN의 중간 layer에서 crop한 얼굴의 feature를 추출하여 attention module에 입력하여 이미지의 변조 여부를 판단
  • 하나의 이미지에서 서로 다른 크기의 feature를 뽑아 다양한 영역 간의 attention 관계를 학습할 수 있도록 함
  • inference 시에는 동일한 비디오에서 샘플링된 세 걔의 이미지에 대해 Hard Voting을 진행하여 최종 판결을 내리도록 해 더 안정적인 예측을 할 수 있도록 함
  • 실험 결과 비디오 당 추출한 프레임 수가 30장을 넘어서면 성능이 비슷하게 수렴하다는 것을 확인했고, 학습 리소스 문제를 감안하여 20장의 이미지를 샘플링

수행 프로세스

  1. 비디오 하나당 20장의 이미지를 샘플링하고 MTCNN을 통해 얼굴 부분을 crop한 후 모델의 입력에 맞도록 resize
  1. 일반화 성능 향상을 위해 data augmentation 진행
    • 가우시안 블러와 가우시안 노이즈 적용
    • Face Blinding을 통해 landmark를 활용하여 안면 일부를 의도적으로 제거
      • 모델이 얼굴의 특정 부분에 지나치게 집중해 중요한 특성을 놓치는 것을 방지하는 효과 → 얼굴의 더 일반적인 특성을 고려하게 됨
  1. 전처리 된 데이터를 classification 모델에 넣어 진짜와 가짜를 구분함

추가 실험

  • epoch 횟수를 다양하게 시도하여 수렴 구간이나 오버피팅을 확인하며 학습 진행

    → 50 epoch 이상에서는 성능이 거의 수렴하는 것을 확인

  • 모델의 backbone을 다른 모델로 수정하려고 시도

    → 시간 부족의 문제로 완료하지 못하였음

  • real 데이터 증강을 여러 방법으로 시도
