소개금융 데이터를 분석하여 창의적인 아이디어를 제안하는 대회

대회 소개

  • 디지털 금융 혁신을 선도할 데이터 기반의 혁신적인 비즈니스 인사이트 및 우수 인재 발굴을 위해 금융과 유통, 통신 등 이종 산업 간의 결합 데이터를 활용한 금융 데이터 경진대회
  • 대회 분석용 데이터를 활용해 창의적인 비즈니스 아이디어 또는 문제해결 방법 제안
  • 총 여섯 가지의 활용 가능 데이터 제공

프로젝트 개요

주제 선정 배경

  • 기존에 제공되는 지역별 경제지표들은 개인의 소비를 나타내지 못한다는 사실에 초점을 맞춤
  • 가계와 개인의 소비 성향을 중심으로 한 지표 생성 제안
  • 가계를 기반으로 한 소비지표를 생성하여 ESG 경영과 접목하여 지역 세분화 아이디어를 제시하고 이를 통해 새로운 서비스를 제안

활용 데이터

활용 데이터 목록

  • 신한카드에서 제공하는 라이프스타일 데이터를 메인으로 사용
  • 추가적인 정보를 위해 투자, 보험, 가맹점 데이터 활용

데이터 전처리

  • 메인 데이터의 지역과 기간을 기주으로 정리
  • 피쳐 엔지니어링과 결측치 처리 진

분석 결과

지표 생성 및 활용 데이터

  • 쇼핑, 자기 투자 등 여러 분야를 선정하여 총 10개의 새로운 지표 생성
  • 지표 생성 시 고려사항
    • 데이터 통합 과정에서 발생하는 지역 간의 차이를 전처리를 통하여 상대적으로 비교가 가능할 수 있게 함
    • 전반적인 소비 활동을 표현하기 위해 다양한 분야에서 지표 생성
    • 지역 대표성 확보를 위해 많은 데이터를 가지고 있는 지표만 사용

지표 생성 방법

  • 정규화: 데이터 scaling
  • 반영 비율: 해당 지역의 지표 데이터의 반영 비율로 변환
  • Entropy Weight: 지표의 속성 정보를 활용하여 가중치 산정
  • 후처리: 지표 생성 후 만들어진 지표를 다시 minmax scaling 진행

지표 설명력 검증

  • 선형회귀: 메인 데이터의 카드 소비액이 소비를 가장 잘 표현할 수 있다고 판단하여 Y값으로 선정
  • R-Squared 점수 하에 약 82.8% 소비를 표현하는 것으로 확인

지역 세분화

  • 클러스터링 이용 → 지표를 구성하는 71개의 피쳐에 대해 진행
    • Hierachical Ward, K-means, DBSCAN 이용
  • 클러스터 수 설정을 위해 실루엣 계수와 Scree plot 시각화
  • K-means를 선택하여 7개의 군집 생성
  • 이후 더 자세한 군집 파악을 위해 Box plot과 Strip Plot으로 클러스터별 특징 파악

아이디어 제안 및 기대효과

  • 다양한 카드사에서 그린카드를 통해 여러 혜택을 제공하고 있음
  • 지역 세분화를 통해 해당 지역에 특화된 혜택을 제공하는 새로운 버전의 그린카드 출시 제안
  • 예를 들어 여행지표가 높은 지역에는 친환경 여행을 다닐 시 숙소비를 할인해주는 혜택을 제공하는 식으로 카드 사용을 장려할 수 있음
  • 적극적인 홍보를 통해 고객 참여를 유도한다면 서비스에 대한 개인의 체감률이 상승해 고객 참여 활성화를 기대할 수 있음
  • 결과적으로 기업의 이미지가 빠른 속도로 제고될 것이며 이는 국내외 수익 제고로 연결되어 기업 수익 향상 또한 기대 가능
